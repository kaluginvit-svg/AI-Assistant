# ИИ-ассистент для компании по изготовлению мебели

![AI Assistant for Furniture Company](https://i.imgur.com/placeholder.png)

## Общее описание проекта

**ИИ-ассистент для ПК «Спецмебельторг»** — интеллектуальный виртуальный помощник, созданный для автоматизации первичной коммуникации с клиентами и оптимизации рабочего времени менеджеров компании. Проект реализован с применением передовых технологий искусственного интеллекта и ориентирован на повышение качества клиентского сервиса при одновременном снижении операционных издержек.

Система способна вести осмысленный диалог с потенциальными клиентами, отвечать на стандартные вопросы о продукции и услугах, а также осуществлять запись на консультации, полностью заменяя рутинные задачи сотрудников первой линии поддержки.

## Цели проекта

- **Снижение нагрузки** на менеджеров по продажам за счёт автоматизации ответов на часто задаваемые вопросы
- **Повышение конверсии** за счет круглосуточной доступности для клиентов
- **Оптимизация процессов** записи клиентов на встречи с менеджерами
- **Стандартизация** ответов по ключевым вопросам для обеспечения единой клиентской коммуникации
- **Сбор аналитики** по запросам клиентов для улучшения продуктовой линейки и сервиса

## Функциональные возможности

### 1. Осмысленное ведение диалога
- Естественное общение на русском языке с пониманием контекста разговора
- Распознавание намерений клиента и подбор релевантных ответов
- Возможность обрабатывать сложные, многоступенчатые запросы
- Адаптация стиля общения под целевую аудиторию (бизнес-клиенты, госзаказчики)

### 2. Работа с базой знаний
- Мгновенные ответы на 50+ часто задаваемых вопросов по продукции и услугам
- Детальная информация о материалах, технологиях производства и гарантиях
- Предоставление сведений о преимуществах сотрудничества с компанией
- Консультирование по специфике мебели для разных типов учреждений

### 3. Запись на встречи
- Сбор контактных данных клиента (имя, телефон, email)
- Предложение и выбор удобного времени для консультации
- Автоматическое добавление встречи в Google Календарь менеджера
- Отправка клиенту подтверждения о записи с деталями встречи
- Напоминание о предстоящей встрече за 24 часа

## Технологическая реализация

### Используемые платформы
- **Савви (https://suvvy.ai/)** — основная платформа для разработки и развертывания ИИ-ассистента с возможностью интеграции в различные мессенджеры и сайты
- **Qwen** — использовался для создания системного промпта и структурирования базы знаний

### Ключевые характеристики платформы Савви:
- Работа на базе надёжных LLM-моделей
- Ответы на 100% обращений клиентов
- Точность ответов от 95%
- Поддержка сложных бизнес-сценариев
- Возможность глубокой кастомизации под специфику компании
- Интеграция с CRM-системами и внешними сервисами

## Структура базы знаний

База знаний включает в себя 7 ключевых разделов:

1. **Описание компании** — информация об опыте работы, производственных мощностях, географии поставок
2. **Услуги** — детализация всех предоставляемых услуг по направлениям:
   - Производство корпусной мебели
   - Дополнительные услуги (доставка, сборка, вывоз мусора)
   - Специализированные решения для разных типов учреждений
3. **Преимущества** — 8 ключевых преимуществ компании перед конкурентами
4. **Контактная информация** — адреса, телефоны, география работы
5. **FAQ (50 вопросов)** — разделён на тематические блоки:
   - Общие вопросы о компании
   - Продукция и её характеристики
   - Услуги и условия сотрудничества
   - Преимущества работы с компанией
   - Гарантийные обязательства
6. **Коммерческие предложения** — шаблоны ответов с ценовыми диапазонами
7. **Сценарии обработки возражений** — готовые ответы для работы со сложными запросами

- Отправка приглашений клиентам по email

### CRM-система (планы на развитие)

- Фиксация результатов взаимодействия для последующей аналитики
